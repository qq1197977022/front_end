一.html

1.生成结构的快捷键

  1. ! + tab,可以生成html5的结构代码。
  2. 生成id名和类名
  3. 标签名.类名#id名+tab
  4. 没有标签名.类名+tab ==>div

2.生成同级元素

  1. 标签名+标签名+标签名 “+”tab

3.生成子类标签

  1. 标签名>子标签名>子标签名>子标签名+tab
  2. 标签名>子标签名>子标签名>子标签名^^子标签名+tab

4.带固定数量的标签

  1. ul>li*5+tab

5.带有序号名称

  1. ul>li.abc$*3 + tab

6.生成带有内容的标签

  1. ul>li>a{item}*5
