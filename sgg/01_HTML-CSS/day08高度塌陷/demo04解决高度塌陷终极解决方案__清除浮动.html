二.总结:

1.浮动:
1.使元素脱离文档流,收回元素在文档流中占据空间位置的使用权(详情: 参见 day07浮动\demo02浮动一)
2.清除浮动
1.重新赋予当前元素left/right/both浮动元素在文档流中的空间位置使用权

三.应用 --- 解决高度塌陷问题

塌陷




解决塌陷

添加一个块级元素,设置清除浮动,来解决高度塌陷问题
弊端:
1.html负责结构,但这里通过添加html元素来解决本该由CSS负责表现问题
优点:
全部兼容

解决塌陷

1.通过CSS解决 --- nice
1.使用伪元素选择器:after选择元素内容之后部分
2.添加内容,相当于添加了一个html元素,清除该元素指定方向(left/right/both)上的浮动
2.与上述通过添加html元素原理一样
1.只是这里是通过CSS完成
3.缺点
IE < 6.0不支持伪元素选择器:after/:before

四.应用总结

1.设置如下CSS样式 
    
        .clear:after{
            clear: xxx;
        }
2.在需要的元素中引入即可
3.兼容IE < 6.0(开启hasLayout隐藏属性) 
    兼容w3c标准浏览器
    
        .clear:after{
            clear: xxx;
        }
    
    兼容IE < 6.0(开启hasLayout隐藏属性)
    
        .clear{
            zoom: 1;
        }
